# Инструкция по проверке notebook

## Автоматическая проверка

Запустите скрипт проверки:
```bash
python3 check_notebook.py
```

## Ручная проверка в Jupyter

### 1. Запуск Jupyter Notebook
```bash
jupyter notebook aircraft_incident_ml.ipynb
```
или
```bash
jupyter lab aircraft_incident_ml.ipynb
```

### 2. Проверка по шагам

#### Шаг 1: Проверка импортов
- Запустите первую ячейку с импортами
- Убедитесь, что все библиотеки установлены
- Если какой-то библиотеки нет, установите:
  ```bash
  pip install pandas numpy matplotlib seaborn scikit-learn
  pip install catboost  # опционально
  pip install tensorflow  # опционально
  ```

#### Шаг 2: Проверка загрузки данных
- Запустите ячейки с загрузкой данных
- Убедитесь, что файл `/Users/nickolay/Downloads/Aircraft_Incident_Dataset.csv` существует
- Проверьте, что данные загрузились корректно

#### Шаг 3: Пошаговое выполнение
- Выполняйте ячейки последовательно (Cell → Run All или Shift+Enter)
- Следите за ошибками в выводе
- Если есть ошибки, проверьте:
  - Правильность путей к файлам
  - Наличие всех необходимых библиотек
  - Корректность данных в датасете

### 3. Возможные проблемы и решения

#### Проблема: "ModuleNotFoundError"
**Решение:** Установите недостающую библиотеку:
```bash
pip install <название_библиотеки>
```

#### Проблема: "FileNotFoundError" при загрузке датасета
**Решение:** Проверьте путь к файлу в ячейке загрузки данных. Убедитесь, что файл существует.

#### Проблема: "MemoryError" или медленная работа
**Решение:** 
- Уменьшите размер датасета для тестирования
- Используйте меньше итераций в GridSearchCV
- Уменьшите количество признаков

#### Проблема: CatBoost или TensorFlow не установлены
**Решение:** Это нормально! Notebook будет работать без них, просто пропустит соответствующие модели.

### 4. Быстрая проверка (только первые шаги)

Для быстрой проверки можно выполнить только первые несколько ячеек:
1. Импорты
2. Загрузка данных
3. Базовый анализ (.info(), .describe())
4. Проверка пропущенных значений

Если эти шаги работают, остальное должно работать тоже.

### 5. Проверка результатов

После выполнения всех ячеек проверьте:
- ✓ Все графики отображаются
- ✓ Модели обучаются без ошибок
- ✓ Метрики вычисляются корректно
- ✓ Выбрана лучшая модель
- ✓ Прогнозы для новых данных работают
